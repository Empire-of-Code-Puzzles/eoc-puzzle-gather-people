We need to be prepared for emergency situations, so it would be good for us to develop a model evacuation process. The rooms in the building are connected by corridors of various lengths (but they can be isolated). Each room may contain a varying number of occupants at any given time. We should calculate how many people can get to the evacuation zone by a given time.

Room and hallway information can be represented in a matrix (M). Each room has a number from 0 to N where the matrix has size of NxN. In the diagonals we are given information about how many people occupy the room -- M[i][i] is how many people in the i-th room.
In other cells of the matrix, you are given an information about the hallways, specifically how much time (in minutes) is required to move from room i to room j (M[i][j] == M[j][i]). If time the is zero, then a hallway does not exist between these rooms. The evacuation zone is a room marked with the number 0 (zero).

You are given a building matrix and evacuation time in minutes. People must always use the shortest path to evacuate the building. By the specified time, how many people will have escaped? For this mission, passing through a room doesnâ€™t take any time, only passing through hallways. A room will be considered evacuated if the total time to evacuate is less than or equal to the given time.
